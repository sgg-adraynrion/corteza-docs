== PDF render options

Render options allow you to specify parameters of the resulting PDF document.
Render options are provided with a variable of the `RenderOptions` type.

Refer to the xref:integrator-guide:templates/index.adoc#_the_preview[menu:Integrator Guide[Templates]] for a complete reference on the available options.
